# Archival notice

oculr-ngx is scheduled to be archived on October 1, 2026.

After that date, no new features, routine dependency updates, Angular version upgrades, bug fixes, or support are planned. The package and source code may remain available under the existing MIT license, but consumers should expect the repository to be read-only and unmaintained.

## What this means for consumers

- New consumers should avoid adopting oculr-ngx unless they are prepared to maintain their own fork.
- Existing consumers should plan a migration path or decide whether to fork the package before October 1, 2026.
- Existing consumers should pin the package version they rely on and continue applying their own security, compatibility, and dependency review processes.
- Teams using oculr-ngx with newer Angular versions should validate compatibility in their own applications.

## Before the archive date

Until October 1, 2026, maintainers expect to focus only on critical security fixes, critical bug fixes, and documentation updates related to the archive. New features, large refactors, and routine dependency updates are unlikely to be accepted.

## After the archive date

After October 1, 2026, no supported versions are planned. Consumers who need updates should migrate away from oculr-ngx or maintain a fork.
